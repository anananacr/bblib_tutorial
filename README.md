# bblib_tutorial

Jupyter notebooks showing examples how to run bblib.

## Depencies:

- Python 3.10
- requirements.txt

## Installation

git clone
pip install -r requirements.txt

## Usage

- Open jupyter-lab

- Open the file methods_import.ipynb

- Run the cells

- Plots will be saved in the center_refinement/plots folder
